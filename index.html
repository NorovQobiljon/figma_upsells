Encourage repeat

Features

Give points for a product, or groups created by taxonomies

You can fully manage the amount of bonuses accrued for the purchase of both individual products and groups. Groups are products united by common taxonomies (Categories, tags, attributes, price, brands) that you can create and customize in an unlimited number.

Or try the free version

Easy-to-manage control of user
points balance

The plugin allows you to conveniently track, accrue and control the balance of bonus points both for each individual user and in bulk according to a certain user attribute. At the same time, for each accrual / write-off,
you can specify a reason, send a newsletter, and much more.

Auto actions for accruing
bonuses

In this section, you can create and configure
automatic points accrual companies based
on a trigger or conditions, followed by user
notification via email or popup.
Set up auto bonuses for registration, birthday,
users of a certain age, amount of purchases,
time since registration, users who abandoned
the cart, users who left a review and many
other triggers and conditions.

Global settings and design

The plugin has the widest customization of each element of the bonus system on the front:

The plugin has the widest customization of each element of the bonus system on the front:

Set the expiration date of certain bonus points.

Set the percentage/amount of the purchase that can be redeemed with bonuses.

Currency, minimum age of participants, usage logic with coupons, shipping fees, refund settings and more.

Moreover, from the specified settings, a page of rules for using the bonus system is automatically generated, which are available to the user in his personal bonus account.
